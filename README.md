# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Sahil91029/Portfolio.git
```

Move into the project

```bash
cd Portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Create a production build

```bash
npm run build
```
